Ref< BitMatrix > HybridBinarizer::getBlackMatrix ( )
virtual

Calculates the final BitMatrix once for all requests.

This could be called once from the constructor instead, but there are some advantages to doing it lazily, such as making profiling easier, and not doing heavy lifting when callers don't expect it.
