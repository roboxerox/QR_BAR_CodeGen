Detailed Description

Decodes Code 39 barcodes. This does not support "Full ASCII Code 39" yet.

Ported form Java (author Sean Owen)

◆ Code39Reader() [1/3]

Code39Reader::Code39Reader ( )

Creates a reader that assumes all encoded data is data, and does not treat the final character as a check digit.

It will not decoded "extended Code 39" sequences.

◆ Code39Reader() [2/3]

Code39Reader::Code39Reader ( bool  usingCheckDigit_)

Creates a reader that can be configured to check the last character as a check digit.

It will not decoded "extended Code 39" sequences.

Parameters
usingCheckDigitif true, treat the last data character as a check digit, not data, and verify that the checksum passes.
