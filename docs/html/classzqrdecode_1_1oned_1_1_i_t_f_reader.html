◆ validateQuietZone()

void ITFReader::validateQuietZone ( Ref< BitArray >  row,
int  startPattern 
)
private

The start & end patterns must be pre/post fixed by a quiet zone.

This zone must be at least 10 times the width of a narrow line. Scan back until we either get to the start of the barcode or match the necessary number of quiet zone pixels.

Note: Its assumed the row is reversed when using this method to find quiet zone after the end pattern.

ref: http://www.barcode-1.net/i25code.html

Parameters
rowbit array representing the scanned barcode.
startPatternindex into row of the start or end pattern.
Exceptions
ReaderExceptionif the quiet zone cannot be found, a ReaderException is thrown.
