202 /*
203  * A QR Code symbol, which is a type of two-dimension barcode.
204  * Invented by Denso Wave and described in the ISO/IEC 18004 standard.
205  * Instances of this class represent an immutable square grid of dark and light cells.
206  * The class provides static factory functions to create a QR Code from text or binary data.
207  * The class covers the QR Code Model 2 specification, supporting all versions (sizes)
208  * from 1 to 40, all 4 error correction levels, and 4 character encoding modes.
209  *
210  * Ways to create a QR Code object:
211  * - High level: Take the payload data and call QrCode::encodeText() or QrCode::encodeBinary().
212  * - Mid level: Custom-make the list of segments and call QrCode::encodeSegments().
213  * - Low level: Custom-make the array of data codeword bytes (including
214  * segment headers and final padding, excluding error correction codewords),
215  * supply the appropriate version number, and call the QrCode() constructor.
216  * (Note that all ways require supplying the desired error correction level.)
217  */
